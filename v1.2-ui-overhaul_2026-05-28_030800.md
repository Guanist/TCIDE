# TCIDE v1.2 UI 全面升级 — Activity Bar + 底部面板 + 增强 AI 界面

**时间**: 2026-05-28 02:53-03:08 CST
**产出**: Activity Bar、底部面板（Terminal/Problems/Output）、增强 AI 面板、标签页拖拽排序、文件树引导线

---

## 概述

基于参考 IDE 截图（Cursor 级体验），对 TCIDE 进行系统性 UI 重构，从基础三栏布局升级为专业 IDE 五区布局。

## 新增 UI 组件

### 1. Activity Bar（活动栏）
- 左侧 48px 固定宽度图标导航栏
- 顶部：资源管理器、搜索
- 底部：设置
- 活跃指示器（左侧橙色竖线）
- SVG 图标 → 无依赖

### 2. 底部面板（Panel Area）
- **终端** (`panel-terminal`)：xterm.js 终端（从编辑器区移至此）
- **问题** (`panel-problems`)：编译错误/警告列表，含图标 + 文件定位
- **输出** (`panel-output`)：构建日志/AI执行日志/主进程日志 通道选择
- 面板拖拽调整器（4px, row-resize, 80-500px 范围）
- 标签页切换（TERMINAL | 问题 | 输出）
- 关闭按钮（Ctrl+J 快捷键）
- 状态栏切换按钮带活跃指示

### 3. AI 面板增强
- **新 Header**：AI 助手标题 + 模型选择器 + 新建对话/对话历史按钮
- **对话历史视图**：可切换聊天列表，显示最近消息摘要，支持删除
- **新版欢迎**：虎猫 AI 图标 + 快速配置按钮 + 提示词芯片
- **输入区域优化**：
  - 单行紧凑输入框（自动扩展至 180px）
  - SVG 附件按钮 + 发送按钮
  - 底部状态栏（当前模型 + 按键提示）
  - 保留原有附件上传功能

### 4. 标签页拖拽排序
- `draggable=true` + `dragstart/dragend/dragover/drop` 事件
- 实时交换位置，drop 后同步 `openFiles` 数组顺序
- 拖拽时半透明 + 指示线

### 5. 文件树缩进引导线
- CSS `::before` / `::after` 绘制树状连接线
- `.level-1` 至 `.level-5` 递进缩进
- `.has-guides` 类控制引导线开关

### 6. 视觉细节
- AI 欢迎页提示词芯片（hover 边框高亮）
- 代码块复制按钮（hover 显示）
- 流式输出光标动画
- 消息时间戳（hover 显示）
- 键盘快捷键录制动画
- 悬停发光效果 `.hover-glow`

## HTML 变更（index.html）
- 新增 `#activity-bar`（SVG 图标导航）
- `#terminal-container` 移入 `#panel-area > #panel-terminal`
- 新增 `#panel-resizer`、`#panel-area`、`#panel-tabs`
- 新增 `#panel-problems`、`#panel-output`
- AI 面板 Header 重构（title-area + header-actions）
- 新增 `#chat-list-view`、`#chat-content-view`
- AI 欢迎区改为 `.ai-welcome-section`
- 新增 `model-select-header`、`.ai-prompt-chip`
- 标签页右键菜单新增「关闭全部」「向右拆分」
- 状态栏 `#status-terminal` → `#status-panel-toggle`

## CSS 变更（main.css）
- `body`：flex column 布局
- `#app`：flex:1（取代 height: 100vh）
- 新增 350+ 行：Activity Bar、Panel、Chat 增强、Tab Drag、引导线、动画

## TS 变更（main.ts）
- `toggleTerminal()`：切换 `#panel-area` + `#panel-resizer`（替代旧 toggle）
- `initTerminal()`：防重复初始化（`terminalInitialized` 标志）
- `renderEditorTabs()`：className `editor-tab` → `tab-item`，添加 draggable
- 新增函数：`syncOpenFilesOrder()`、`renderChatList()`
- 新增事件：Activity Bar、Panel Tabs、Panel Resizer、Tab Drag、AI Header
- 新增 `tab-close-all`、`tab-split-right` 上下文菜单处理
