# Task Artifact: TCIDE v1.1 Phase 2 — UI/UX 全面优化

**时间**: 2026-05-27 23:08-23:40 CST
**目标**: 融合 Trae / Claude Code / Codex / QClaw / CodeBuddy 优点，系统优化 UI 和用户体验

## 参考产品设计模式

| 产品 | 借鉴的设计理念 |
|------|---------------|
| **Trae** | 暗色主题渐变点缀、标签页滑动指示器、按钮微交互、焦点环动画 |
| **Claude Code** | Agent 进度可视化、结构化输出、权限确认流程 |
| **Codex/CodeBuddy** | 行内补全反馈、快速操作面板、模型状态可视化 |
| **QClaw** | 文件式记忆系统、设计令牌体系、全局过渡一致性 |

## 设计系统升级

### CSS Variables 全面扩展（+30 个新令牌）
- **间距比例**: `--space-xs` ~ `--space-2xl` (4px~24px)
- **圆角比例**: `--radius-sm` ~ `--radius-full` (4px~999px)
- **阴影层级**: `--shadow-sm` ~ `--shadow-xl` + `--shadow-glow`
- **缓动函数**: `--ease-fast/normal/smooth/bounce` (cubic-bezier 标准化)
- **字体系统**: `--font-sans/mono` + 8 级字号
- **玻璃背景**: `--bg-glass`, `--bg-glass-hover`
- **渐变色**: `--tc-gradient` (虎猫橙渐变)
- **文字色**: `--text-dim` (低强调文字)

### 全局交互增强
- 按钮按下缩放 `scale(0.97)`
- `focus-visible` 橙色焦点环
- 文本选中橙色高亮
- 字体抗锯齿渲染
- 全局滚动条一致性 (Firefox + WebKit)

## 组件级优化详情

### 1. 编辑器标签页 (Trae 风格)
- **滑动指示器**: `::after` 伪元素 + `scaleX` 过渡动画
- **关闭按钮**: 默认隐藏，hover/active 时淡入，hover 时变红
- **修改标记**: 通过 `.tab-name::after` 添加橙色圆点
- **圆角边缘**: 右侧 4px 圆角，视觉更柔和

### 2. 状态栏
- 渐变背景 `linear-gradient(135deg, #CC7000, #B8620A)`
- 悬停时半透明白色背景
- 圆角项目，更好视觉层次

### 3. AI 对话面板
- **模型状态指示器**: 头部显示模型名 + 连接状态圆点
- **打字指示器**: 3 个弹跳圆点，AI 响应时显示
- **消息气泡**: 用户消息渐变背景，助手消息边框包裹
- **角色标签**: hover 时显示 sender 信息
- **消息入场动画**: `msgSlideIn` 从下方滑入

### 4. 右键菜单
- `scaleIn` 缩放入场动画
- `backdrop-filter: blur(8px)` 毛玻璃效果
- 更大间距 + 快捷键显示支持

### 5. 设置面板（卡片式布局）
- 每个配置区独立卡片 (`.settings-section`)
- 输入框焦点发光环 `box-shadow: 0 0 0 3px var(--tc-glow)`
- 保存按钮渐变 + hover 上浮 + 发光
- 表单提示文本 (`.form-hint`)

### 6. 空状态设计
- 图标 + 标题 + 描述 + 操作按钮四段式
- 应用于文件树、任务列表、大纲面板

### 7. 加载骨架屏
- `shimmer` 动画 (渐变平移)
- 三种变体: `.skeleton-line`, `.skeleton-block`, `.skeleton-circle`

### 8. 欢迎页特征卡片
- 更大内边距 (12→16px)
- 悬停上浮 3px + 更强发光
- 字号和间距优化

### 9. 任务列表
- 运行中任务内发光边框
- 错误任务红色背景
- 空状态图标引导
- 等宽字体任务 ID

### 10. 文件树
- 右侧圆角，玻璃悬停效果
- 活动项白色文字
- 搜索框优化

### 11. 聊天输入区
- 文本框增大圆角 + 焦点发光环
- 悬停时边框变亮
- 发送按钮渐变 + hover 上浮 + 发光
- 发送按钮禁用态
- 终止按钮红色发光

### 12. 拖拽调整器
- hover 时显示细线指示器
- 拖动时橙色发光背景

### 13. 全局按钮交互
- 统一 `transition` 属性组
- `:active` 缩放反馈
- `:focus-visible` 橙色环
- 渐变主按钮变体

## 功能新增
