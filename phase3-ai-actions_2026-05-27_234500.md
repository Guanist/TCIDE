# Task Artifact: TCIDE v1.1 Phase 3 — 编辑器 AI 快捷操作 + 配置管理

**时间**: 2026-05-27 23:29-23:45 CST
**目标**: 编辑器右键 AI 快捷操作、设置导入导出、文件树上下文菜单增强

## 新增功能

### 1. 编辑器右键 AI 快捷操作（4 个新 Action）

从 Claude Code 的 Agent Tool 模式借鉴，将 AI 能力直接集成到编辑器右键菜单：

| Action | ID | 功能 | 提示词策略 |
|--------|-----|------|-----------|
| **AI 重构代码** | `tcide-ai-refactor` | 提高可读性和可维护性 | 指定语言，要求直接返回代码 |
| **AI 生成测试** | `tcide-ai-tests` | 生成完整单元测试 | 覆盖主要功能和边界情况 |
| **AI 修复 Bug** | `tcide-ai-fix` | 修复 Bug/性能/安全 | 三类问题一起检查 |
| **AI 生成文档** | `tcide-ai-docs` | 生成详细文档注释 | 包含参数和返回值说明 |

- 全部可通过 **右键菜单**（`tcide-ai` 组）和 **命令面板** (Ctrl+Shift+P) 调用
- 复用现有 `aiGenerateAndInsert` 流程 → 流式输出 → Diff 预览 → 接受/拒绝
- 语言感知：prompt 中注入当前文件语言类型

### 2. 设置导入/导出

- **导出配置**: 保存为 `tcide-config.json`，不导出 API Key（安全）
- **导入配置**: 选择 JSON 文件，自动应用并保存（保留当前 API Key）
- 完整 IPC 链路: `config:export` / `config:import` → Electron dialog → 文件读写

### 3. 文件树上下文菜单增强

新增两个操作：
- **📄 新建文件** — 在当前目录或选中目录下创建文件
- **📁 新建文件夹** — 创建目录（IPC `file:mkdir`，支持递归）

菜单重新组织为三组：创建 → 修改 → 工具，视觉更清晰。
