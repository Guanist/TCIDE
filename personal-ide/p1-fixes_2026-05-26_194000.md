### Anthropic Messages API:
- Endpoint: `POST {baseUrl}/messages`
- Auth: `x-api-key: {key}` + `anthropic-version: 2023-06-01`
- System prompts extracted to top-level `system` field (Anthropic convention)
- Non-streaming: parses `content[{type:"text", text:"..."}]`
- Streaming: parses SSE events `content_block_delta`, `message_start`, `message_delta`
- Token usage: `usage.input_tokens` / `usage.output_tokens`

## P1-2: Request Retry with Exponential Backoff

**Method**: `withRetry()` wrapper in `ModelAdapter.send()`

- Retryable: HTTP 429, 500, 502, 503, 504 + rate_limit/overloaded in body
- Backoff: 1s → 2s → 4s (max 3 retries = 4 total attempts)
- Non-retryable: 4xx errors (except 429), AbortError (user cancel)
- Logs each retry to console
