## 改动 2: 多会话对话系统

### 数据结构
```typescript
interface ChatSession {
  id: string;
  name: string;           // 自动用首条用户消息命名
  chatHistory: ChatMessage[];
  createdAt: number;
  updatedAt: number;
  projectPath?: string;
}

state.chatSessions: ChatSession[]
state.currentSessionId: string
```

### 功能
| 操作 | 说明 |
|------|------|
| `ensureSession()` | 确保当前有活跃会话，没有则创建 |
| `createSession(name?)` | 创建新会话，插入列表首部 |
| `switchSession(id)` | 切换会话 → 清空消息区 + 重新渲染该会话历史 |
| `deleteSession(id)` | 删除会话，自动切到下一个 |
| `renderChatList()` | 渲染会话列表（标题 = 首条用户消息 / 消息数 / 时间） |

### UI 交互
- Header `+` 按钮 → 新建空会话
- Header `☰` 按钮 → 打开/关闭对话列表
- 列表项点击 → 切换会话
- 列表项 `×` → 删除会话
- 首条消息发送时自动命名会话
