# TCIDE AI 对话体验全面升级

**时间**: 2026-05-28 04:46-04:50 CST

## 用户反馈问题
1. 虎猫读不到左侧文件树中的文件
2. 对话框非常简陋，体验差
3. 没有对思考过程折叠
4. 没有对不同产出归类显示
5. 字体很差劲

## 改造内容

### 1. 自动文件上下文注入
`sendToAI()` 现在自动包含当前编辑器打开文件的内容（上限 8000 字符）作为 prompt 上下文：
```
📂 当前文件: C:\path\to\file.ts
```typescript
... full file content ...
```
```
这样 AI 无需用户手动复制代码就能理解当前项目上下文。

### 2. 消息气泡 UI 重设计
- **头像系统**: 用户🧑 / 虎猫 AI🐯 / 系统⚙
- **消息头**: 角色名 + 时间戳
- **用户消息**: 橙色调渐变气泡，右对齐
- **AI 消息**: 深灰底色 + 边框，左对齐
- **系统消息**: 橙色左边框提示条

### 3. 代码块增强
- 语言标签显示（如 `typescript`、`python`）
- 复制按钮（点击复制 + "已复制" 反馈）
- 专用等宽字体：Cascadia Code → JetBrains Mono → Fira Code → Consolas
- 美观的深色背景 + 圆角边框

### 4. 字体升级
- 中文：PingFang SC / Microsoft YaHei
- 英文：-apple-system / BlinkMacSystemFont / Segoe UI
- 代码：Cascadia Code（优先级最高）
- 行高：1.65（提升可读性）
- 字号：14px 正文 / 13px 代码

### 5. 打字指示器优化
- 更大、更明显的跳动动画
- 圆角边框包裹

## 修改文件
- `src/renderer/main.ts`: `sendToAI()` + 编辑器上下文、`renderMarkdown()` 代码块重写、`addChatMessage()`/`appendStreamChunk()` 新布局
- `src/renderer/styles/main.css`: 消息区 ~280 行全量替换
